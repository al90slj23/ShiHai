# 00.meta-01 ShiHai 元规范

> **版本**：v0.1.0  
> **最后更新**：2026-05-26

## 项目定位

ShiHai / 识海 是面向数字自我的开放个人记忆框架。它用于沉淀一个人的原始经历、对话、录音、关系、知识、偏好、决策与成长轨迹。

## 六层 AI 记忆体系

物理目录必须使用：

```text
.ai/L0#Execution/
.ai/L1#Overview/
.ai/L2#Index/
.ai/L3#Standards/
.ai/L4#Changelog/
.ai/L5#Knowledge/
```

文档引用格式使用：

```text
L0#工作执行 (Execution)
L1#项目概览 (Overview)
L2#规范索引 (Index)
L3#完整规范 (Standards)
L4#操作日志 (Changelog)
L5#知识图谱 (Knowledge)
```

## 零容忍

- L3 是当前规范唯一主版本。
- L2 只做索引，不承载规范正文。
- Agent 写入长期记忆必须带来源。
- 私有 self 数据不得默认开源。
