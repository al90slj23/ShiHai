# 01.arch-01 ShiHai 核心架构

> **版本**：v0.1.0  
> **最后更新**：2026-05-26

## 根目录职责

`ShiHai/` 根目录存放通用框架层：

```text
docs/
schemas/
templates/
tools/
workflows/
protocols/
agents/
examples/
```

这些内容应尽可能与具体个人无关，未来可以开源给其他人使用。

## Self 实例职责

`selves/<SelfName>/` 存放一个具体数字自我的全部差异化记忆数据。

当前实例：

```text
selves/LikeHeng/
```

## 隔离原则

- 框架层可开源。
- self 层默认私有。
- 示例数据放 `examples/`，不要使用真实个人数据。

## 原型实验原则

LikeHeng 是 ShiHai 的第一个 reference self / dogfood instance。ShiHai 的通用能力应优先在 LikeHeng 的真实使用中被验证，再抽象进入框架层。

实践路径：

```text
LikeHeng 真实需求
  → self 内试验
  → 形成稳定流程
  → 抽象为通用 schema/protocol/workflow/template/agent adapter
  → 进入 ShiHai 框架层
```

该原则保证 ShiHai 不是空想架构，而是被真实个人数字记忆建设持续驱动的开源框架。
