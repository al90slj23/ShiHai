# 02.memory-01 Canonical Memory Model

> **版本**：v0.1.0  
> **最后更新**：2026-05-26

## 核心对象

### Event

事件记录发生过什么，必须包含时间、来源、摘要、参与者和置信度。

### Entity

实体包括 person、organization、project、place、concept、object。

### Relation

关系连接两个实体，必须带证据和置信度。

### Claim

断言是 AI 或用户认为成立的事实、偏好、目标、信念或假设。AI 推断型 claim 应支持 `needs_review`。

## Review Queue

敏感记忆先进入：

```text
03_Canonical/review_queue/pending_memory.jsonl
```

审核后再进入 canonical claims / relations / knowledge profiles。

## 必要元数据

```json
{
  "created_by": "hermes|openclaw|other",
  "created_at": "ISO-8601",
  "source_ref": "path or external reference",
  "confidence": 0.9,
  "needs_review": true
}
```
