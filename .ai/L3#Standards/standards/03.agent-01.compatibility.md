# 03.agent-01 Agent Compatibility

> **版本**：v0.1.0

## 核心原则

ShiHai 不属于任何单一 Agent。Hermes、OpenClaw、Claude、Codex、Obsidian、mem0、Graphiti 都只是客户端或组件。

## Hermes

- 来源标记：`.h`
- 通用适配：`agents/hermes/`
- LikeHeng 个性化上下文：`selves/LikeHeng/07_Agents/hermes/`

## OpenClaw

- 来源标记：`.c`
- 通用适配：`agents/openclaw/`
- LikeHeng 个性化上下文：`selves/LikeHeng/07_Agents/openclaw/`

## 接入协议优先级

1. MCP server
2. File protocol
3. Agent context pack
4. HTTP API

## 写入原则

Agent 不应直接覆盖长期人物/身份/关系档案。优先写入 review queue，由用户或可信流程审核合并。
