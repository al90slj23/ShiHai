# 04.privacy-01 Open Source Boundary

> **版本**：v0.1.0  
> **最后更新**：2026-05-26

## 可开源

- `docs/`
- `schemas/`
- `templates/`
- `tools/`
- `workflows/`
- `protocols/`
- `agents/`
- `.ai/`
- `examples/` 中的脱敏示例

## 默认不可开源

- `selves/LikeHeng/`
- 原始录音、聊天、截图、PDF
- 人物关系档案
- 身份、价值观、健康、财务、情绪模式
- Agent 写入日志中的敏感来源

## Git 规则

`.gitignore` 必须默认忽略：

```text
selves/*
!selves/.gitkeep
```

如果需要提交示例 self，必须放入 `examples/` 并脱敏。
